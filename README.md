# Sentiment Analysis System

A Django web application for analyzing customer reviews using machine learning.

## Features
- Single review sentiment analysis
- Bulk file processing (CSV/Excel)
- Interactive dashboard
- REST API

## Setup
1. Clone: `git clone https://github.com/dafam0094/sentiment-analysis-system.git`
2. Install: `pip install -r requirements.txt`
3. Run: `python manage.py runserver`
